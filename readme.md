## BTSWA
Blue Team Swiss Army Knife.
# desgined for linux users
install Dependencies (providing you have snap)
```sh
sudo snap mapscii
```

for Python
```sh
python3 btswa.py
```
for C++
```sh
G++ btswa.cpp
./a.out
```
